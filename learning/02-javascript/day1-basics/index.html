变量、数据类型、运算符

📚 变量声明：var / let / const

关键字 作用域 可重新赋值 推荐
var 函数作用域 ✅ ❌ 不推荐
let 块级作用域 ✅ ✅ 变量
const 块级作用域 ❌ ✅ 常量

📊 数据类型

原始类型 (7种)：

  • string - 字符串
  • number - 数字（包括整数和浮点数）
  • boolean - 布尔值 true/false
  • null - 空值
  • undefined - 未定义
  • symbol - 符号（ES6）
  • bigint - 大整数（ES2020）

引用类型：

  • object - 对象（包括数组、函数）

🔄 类型转换

// 转为字符串
String(123)      // "123"
(123).toString() // "123"
123 + ""         // "123"

// 转为数字
Number("123")    // 123
parseInt("123")  // 123
parseFloat("3.14") // 3.14

// 转为布尔值
Boolean(1)       // true
Boolean(0)       // false
!!123            // true

⚠️ 重要：== vs ===

// 记住：永远使用严格相等 ===
5 == "5"   // true  （自动转换类型）
5 === "5"  // false （不转换类型）

// 对象比较
{} === {}   // false （引用不同）
[] === []   // false
💡 最佳实践：
  • 优先使用 const，只有需要重新赋值时用 let
  • 永远不要用 var
  • 比较时使用 === 而非 ==
  • 使用 typeof 检查类型，但注意 null 返回 "object"
