GET POST PUT DELETE PATCH HEAD OPTIONS

Host: api.example.com
User-Agent: Mozilla/5.0

2. HTTP 状态码

200 OK - 请求成功
201 Created - 创建成功
204 No Content - 无内容
301 Moved Permanently - 永久重定向
302 Found - 临时重定向
304 Not Modified - 缓存有效
400 Bad Request - 请求错误
401 Unauthorized - 未认证
403 Forbidden - 无权限
404 Not Found - 未找到
500 Internal Server Error
502 Bad Gateway
503 Service Unavailable

4. HTTPS TLS 握手

1. 客户端 → 服务端：ClientHello
   - 支持的 TLS 版本
   - 加密算法套件
   - 随机数

2. 服务端 → 客户端：ServerHello
   - 选定 TLS 版本
   - 选定加密算法
   - 随机数
   - 服务器证书（包含公钥）

3. 客户端验证证书
   - 检查证书是否由可信 CA 签发
   - 检查证书是否过期
   - 检查域名是否匹配

4. 客户端 → 服务端：Pre-Master Secret
   - 用服务端公钥加密后发送

5. 双方计算会话密钥
   - Master Secret → Session Key

6. 加密通信开始 ✅
