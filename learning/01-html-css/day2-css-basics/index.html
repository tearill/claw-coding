2. 盒模型 (Box Model)

content-box
默认: width=200 + padding=20×2 + border=5×2 = 250px
border-box
width=200 包含 padding + border
最佳实践： 建议在 CSS 开头使用 * { box-sizing: border-box; } 让盒模型更可控
