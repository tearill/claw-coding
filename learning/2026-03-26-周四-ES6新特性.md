### 2. ES6+ 新特性学习

#### 2.1 let/const 与解构赋值

**let**：
- 块级作用域，不同于 var 的函数作用域
- 不存在变量提升（Temporal Dead Zone）
- 不可重复声明

**const**：
- 声明常量，初始化后不可修改
- 对于对象和数组，只保证引用不变，内容可以修改
- 声明时必须初始化

**解构赋值**：
- 数组解构：按位置匹配
- 对象解构：按属性名匹配
- 默认值：当值为 undefined 时使用
- 剩余参数：...rest

```javascript
// 数组解构
const [first, second, third] = [1, 2, 3];

// 对象解构
const { name, age } = person;

// 默认值
const [a = 10, b = 20] = [1];

// 剩余参数
const [head, ...tail] = [1, 2, 3, 4];
```

#### 2.2 模板字符串

- 使用反引号 `` ` `` 包裹
- 支持 `${}` 嵌入表达式
- 支持多行字符串

```javascript
const greeting = `Hello, my name is ${name}, I'm ${age} years old.`;
```

#### 2.3 箭头函数

- 语法简洁：`const add = (a, b) => a + b;`
- 不绑定自己的 this，继承外层作用域的 this
- 不能用作构造函数（不能 new）
- 没有 arguments 对象

```javascript
// 简洁写法
const add = (a, b) => a + b;

// 代码块写法
const multiply = (a, b) => {
  return a * b;
};

// this 绑定
function Timer() {
  this.time = 0;
  setInterval(() => {
    this.time++; // this 指向 Timer 实例
  }, 1000);
}
```

#### 2.4 展开运算符

- 数组展开：`[...arr1, ...arr2]`
- 对象展开：`{ ...obj1, ...obj2 }`
- 函数参数展开：`fn(...args)`

```javascript
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const combined = [...arr1, ...arr2]; // [1, 2, 3, 4, 5, 6]

const obj1 = { a: 1 };
const obj2 = { ...obj1, b: 2 }; // { a: 1, b: 2 }
```

#### 2.5 Promise 与 async/await

**Promise**：
- 三种状态：pending、fulfilled、rejected
- 链式调用：.then().catch()
- Promise.all、Promise.race

**async/await**：
- Promise 的语法糖
- 错误处理用 try/catch

```javascript
const promise = new Promise((resolve, reject) => {
  // 异步操作
  resolve('success');
});

async function fetchData() {
  try {
    const result = await promise;
    return result;
  } catch (error) {
    console.error(error);
  }
}
```

#### 2.6 类 (class)

- ES6 正式引入 class 语法
- constructor 构造函数
- extends 继承
- super 调用父类

```javascript
class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    console.log(`${this.name} makes a sound.`);
  }
}

class Dog extends Animal {
  constructor(name, breed) {
    super(name);
    this.breed = breed;
  }
  speak() {
    console.log(`${this.name} barks: Woof!`);
  }
}
```

#### 2.7 Symbol

- 创建唯一标识符
- 用于对象属性键，避免属性名冲突
- 不会与其他属性名相等

```javascript
const sym = Symbol('description');
const sym2 = Symbol('description');
sym === sym2; // false
```

#### 2.8 迭代器与生成器

**迭代器 (Iterator)**：
- 定义 next() 方法，返回 { value, done }
- 用于 for...of 遍历

**生成器 (Generator)**：
- function* 声明
- yield 暂停执行
- 懒加载序列

```javascript
function* numberGenerator() {
  yield 1;
  yield 2;
  yield 3;
}

const gen = numberGenerator();
gen.next().value; // 1
```

#### 2.9 可选链 (?.) 与空值合并 (??)

**可选链**：
- 安全访问深层属性
- 属性不存在返回 undefined

**空值合并**：
- ?? 只在值为 null/undefined 时使用默认值
- 与 || 的区别：|| 会对 0、''、false 触发

```javascript
user?.profile?.name; // 可选链
value ?? 'default';  // 空值合并
```

### 核心知识点总结

1. **let/const**：块级作用域常量/变量声明
2. **解构赋值**：从数组/对象中提取值
3. **模板字符串**：带表达式的字符串
4. **箭头函数**：简写 + this 绑定
5. **展开运算符**：合并数组/对象
6. **Promise**：异步编程解决方案
7. **async/await**：Promise 语法糖
8. **class**：面向对象语法
9. **Symbol**：唯一标识符
10. **Iterator/Generator**：可迭代协议
11. **可选链/空值合并**：安全访问

### 重要特性对比

| 特性 | 说明 | 示例 |
|------|------|------|
| var | 函数作用域、变量提升 | `var a = 1;` |
| let | 块级作用域、无提升 | `let a = 1;` |
| const | 块级作用域、不可修改 | `const PI = 3.14;` |
| \|\| | 或运算，0 空字符串也触发 | `value \|\| 'default'` |
| ?? | 空值合并，只 null/undefined 触发 | `value ?? 'default'` |

## 问题与反思

### 遇到的问题

1. **空值合并 vs 或运算符**：一开始没理解 `??` 和 `||` 的区别，需要注意 0 和空字符串的场景
2. **箭头函数 this**：虽然知道箭头函数不绑定 this，但有时会忘记，在回调中容易出错
3. **const 对象属性**：以为 const 声明的对象完全不能改，其实只是引用不能改
