## 📚 学习内容

### 1. React 简介

React 是 Facebook 开发的一个用于构建用户界面的 JavaScript 库，具有以下核心特点：

- **组件化**：通过组件来构建 UI，每个组件维护自己的状态和逻辑
- **声明式**：告诉 React 想要什么结果，而不是如何去做
- **单向数据流**：数据从父组件向下流向子组件
- **虚拟 DOM**：在内存中维护一个 DOM 树的副本，提高渲染效率

### 2. JSX 语法

JSX 是 JavaScript 的语法扩展，让我们能在 JavaScript 中写 HTML 一样的语法。

**基本规则**：
1. 必须只有一个根元素（可以用 `<div>` 或 React.Fragment 包裹）
2. 标签必须闭合（自闭合如 `<img />`）
3. class 改为 className
4. for 改为 htmlFor
5. 属性名采用驼峰命名（crossOrigin, tabIndex 等）
6. 布尔属性可以只写属性名（disabled, checked）

### 3. JSX 与 JavaScript 的关系

JSX 只是 `React.createElement()` 的语法糖，最终会被编译为 JavaScript。

```jsx
// JSX 写法
const element = <h1>Hello, World!</h1>;

// 编译后
const element = React.createElement('h1', null, 'Hello, World!');
```
