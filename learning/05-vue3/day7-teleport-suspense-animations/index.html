Vue 3 进阶特性

1. Teleport 传送门

Teleport 是 Vue 3 提供的一个内置组件，可以将组件的模板部分渲染到 DOM 中的指定位置，而不是当前组件的位置。

📌 核心作用：将子组件渲染到父组件 DOM 结构之外的指定位置

使用场景

  • 模态框 (Modal)
  • 下拉菜单 (Dropdown)
  • 工具提示 (Tooltip)
  • Toast 通知

语法

<Teleport to="body">
  <div class="modal">内容</div>
</Teleport>

实际演示

点击按钮打开模态框（使用 Teleport 传送到 body）

提示：打开后查看 Elements 面板，modal 会在 <body> 或 #app 外

禁用 Teleport

<Teleport to="body" :disabled="isDisabled">
  <div>内容</div>
</Teleport>

2. Suspense 异步组件

Suspense 是一个内置组件，用于在异步组件等待时展示 fallback 内容。

📌 核心作用：优雅处理异步组件的加载状态

使用场景

  • 异步组件（setup 返回 Promise）
  • async setup() 组件
  • 带有顶层 await 的 setup()

语法

<Suspense>
  <template #default>
    <AsyncComponent />
  </template>
  <template #fallback>
    <Loading />
  </template>
</Suspense>

实际演示

模拟异步组件加载
⚠️ 注意：Suspense 目前是实验性 API，主要用于 Vue 3 的新特性配合

📝 总结

  • Teleport: 将组件渲染到指定 DOM 位置，适合模态框等场景
  • Suspense: 处理异步组件的加载状态，需要配合异步组件使用
  • Transition: 为元素的插入/移除添加动画效果
  • TransitionGroup: 为列表添加动画，支持移动过渡
💡 提示：Vue 3 的动画系统基于 CSS transition，配合 JavaScript 钩子可以完成更复杂的效果
