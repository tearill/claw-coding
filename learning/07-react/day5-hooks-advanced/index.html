🐈‍⬛ React Hooks 进阶

useRef / useCallback / useMemo 演示

1. useRef 基础 - DOM 引用

const inputRef = useRef(null);
const inputRef = useRef(null);

// 使用
<div ref={inputRef}>...</div>

// 获取 DOM
inputRef.current.focus();
console.log(inputRef.current.value);

2. useRef 变量引用 - 跨渲染保持数据

const countRef = useRef(0);
// 修改不触发重新渲染
countRef.current += 1;
console.log(countRef.current);
点击次数(不渲染): 0

3. useCallback - 记忆化回调函数

// 依赖变化时更新函数引用
const memoizedCallback = useCallback(() => {
  setCount(c => c + 1);
}, []); // 空依赖：函数永远不变

const callbackWithDeps = useCallback((x) => {
  return x * 2;
}, [factor]); // factor 变化时更新
计数器: 0

4. useMemo - 记忆化计算值

// 依赖变化时重新计算
const expensiveValue = useMemo(() => {
  return items.filter(...).sort(...);
}, [items, filter]);
计算复杂度：0

6. 综合示例：计数器

// 使用所有 Hooks 的综合示例
const [count, setCount] = useState(0);
const countRef = useRef(0);
const doubleCount = useMemo(() => count * 2, [count]);
const handleIncrement = useCallback(() => {
  setCount(c => c + 1);
  countRef.current += 1;
}, []);
