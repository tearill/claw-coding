## Part 2: A股金融分析学习路径

### 目标
成为顶尖的金融分析大师，尤其擅长 A股短线交易（龙头战法）

### 每日任务 ⏰

| 时间 | 任务 |
|------|------|
| **交易日 09:00** | 盘前：获取竞价数据、板块资金流向 |
| **交易日 14:30** | 午后：分析盘面、准备复盘 |
| **交易日 15:00** | 收盘：复盘当日操作、记录心得 |
| **每日** | 学习金融知识、分析案例 |

### 学习内容 📖

#### 基础阶段 (Week 1-4)
- [ ] A股交易规则详解（T+1、涨跌停、集合竞价）
- [ ] 龙头战法核心逻辑（什么是龙头、如何识别）
- [ ] 基础技术分析（K线、均线、成交量）
- [ ] 复盘指标学习（资金流向、情绪周期）

#### 技术分析 (Week 5-8)
- [ ] K线形态识别（锤子线、吞没、十字星）
- [ ] 成交量分析（量价关系、量能萎缩）
- [ ] 均线系统（MA5/MA10/MA20/MA60）
- [ ] 板块轮动规律
- [ ] 竞价分析（竞价量、竞价幅度）

#### 实战策略 (Week 9-12)
- [ ] 龙头股筛选条件
- [ ] 买入时机判断
- [ ] 仓位管理（单票仓位、总仓位）
- [ ] 止损纪律（-5% 无条件卖出）
- [ ] 情绪周期识别
- [ ] 退潮期风险控制

#### 量化入门 (Week 13-16)
- [ ] Python 金融分析基础
- [ ] Pandas 数据处理
- [ ] 回测框架入门
- [ ] 简单策略编写
